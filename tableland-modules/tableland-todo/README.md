The initial goal of this project is to develop a task manager platform for FVM smart contract designers using Tableland. Utilizing the tableland to do architecture for development.

```bash
# install dependencies
$ npm install

# set validator address, default is http://localhost:8545
$ export VALIDATOR=http://tableland.com

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate

# after generate serve the static project locally
$ npx nuxt start
```
